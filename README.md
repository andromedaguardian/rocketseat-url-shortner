# URL Shortener
A project created as part of the free Online Java Course at [Rocketseat](https://app.rocketseat.com.br/).

The goal of this project is to implement an "url shortener".

### Used Technologies
- Java
- Maven
- AWS Lambda
- AWS S3
- AWS API Gateway
